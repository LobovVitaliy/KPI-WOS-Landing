BUG BOOKING

Bug Booking allows one to book and list property in short time and with comfort.

Read more

DESCRIPTION

Bug Booking is a web-oriented program system that will allow one to book/host hotels or other property. The system will include 3 types of separate users: Guest – the one who books property, Host – the one who lists property, Admin – the one who supports the service and verifies property if needed. The system enables user with an ability to review and search through the listed property. Different filters (such as location, check-in/out dates, rooms quantity, rent price etc.) are also available. A user is also provided with the lists of most popular/discount locations, and also recommended booking places according to the user’s activity. Every property includes the following information: name, address, quality, users’ rank, photos, map location link, closest landmarks, payment types etc.

Moreover, there is a review section on each property page where one can leave commentaries. The booking function becomes available after a user is signed up and signed in. During the booking process the following data must be provided: special requests (if needed), personal information (includes email address and mobile number), payment details (first/last names, card number, expiry date, security code, billing address). The system also lets one to host his property. In order to publish his property, a host must submit an application the is to be verified by the admin. After the verification the property is published on the website, and the host is now able to edit its information if needed. Admin is the one who performs property verification. He also can add/remove published property, a guest’s/host’s account.

KEY FEATURES OF BUG BOOKING

1

Real-time booking from any place via web or mobile app

2

Real reviews from other guests

3

Ability for hosts to find guests and earn remotely

4

Only verified properties and hosts
